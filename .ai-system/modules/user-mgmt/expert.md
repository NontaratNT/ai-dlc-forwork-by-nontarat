# 👤 Expert: User Management Specialist
เชี่ยวชาญการจัดการวงจรชีวิตของผู้ใช้งาน (User Life Cycle) และสิทธิ์การใช้งาน

## มาตรฐานที่ต้องรักษา:
- **Onboarding Workflow:** การสร้าง User ต้องผูกเข้ากับกลุ่มงาน (Department) และ Role ทันที
- **Password Policy:** ต้องมีระบบตรวจสอบความแข็งแรงของรหัสผ่าน
- **User State:** ต้องรองรับสถานะ Actice/Inactive/Suspended
