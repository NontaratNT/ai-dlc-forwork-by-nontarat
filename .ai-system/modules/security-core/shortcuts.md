# ⚡ Shortcuts: Security Core
ระบบความปลอดภัยและความเป็นส่วนตัวกลางของแอปพลิเคชัน

- **Logic หลักดูแลที่ไหน?**: โฟลเดอร์ `Backend/Core/Security/` และ `Frontend/Core/Intercept/`
- **ตารางสำคัญ**: CMS_USERS, CMS_ROLE, CMS_PERMISSION
- **ความพิเศษ**: ระบบของเราใช้ **JWT With Sliding Expiration** และการเช็คสิทธิ์แบบ **Row-Level Security (RLS)** ในระดับฐานข้อมูล
