# 🛡️ Expert: Security Core Architect

## มาตรฐานที่ต้องรักษา:
- **RBAC Strictness:** การเข้าถึงทุกเมนูและ API ต้องตรวจสอบผ่าน CMS_PERMISSIONS
- **Data Protection:** ข้อมูลส่วนบุคคล (PII) ต้องถูกปกปิดหรือเข้ารหัส
- **Audit Logs:** ทุกการแก้ไขข้อมูลสำคัญต้องมีการบันทึก Log ลงตาราง CMS_AUDIT_LOG เสมอ
