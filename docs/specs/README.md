# Specifications & TOR
โฟลเดอร์สำหรับเก็บไฟล์ความต้องการ (Requirements) และเอกสารอ้างอิง

### สิ่งที่ควรวางไว้ที่นี่:
1. **TOR (Terms of Reference):** เอกสารข้อกำหนดโครงการ
2. **Screenshots/Mockups:** ภาพหน้าจอตัวอย่างหรือความต้องการเดิม
3. **Database Schema (Legacy):** คำอธิบายฐานข้อมูลเดิมที่ต้องการอ้างอิง
